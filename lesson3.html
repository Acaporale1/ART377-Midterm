Let's start organizing our thoughts...



  • Every day we use 'LISTS'.
    • We might make a list for when it's time to buy groceries, or maybe to write down to-do tasks!
  • There are 2 types of LISTS.
    1. An 'ORDERED LIST' is when each list element is preceded with a number (1,2,3,etc,.)
    2. An 'UN-ORDERED LIST' is when each list element is preceded with a non-numeric glyph, such as a bullet point or an arrow.
