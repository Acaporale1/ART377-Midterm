Let's start styling!...



  • It's time to put some personality into our webpages using 'CSS'.
    • 'CSS' stands for CASCADING STYLE SHEETS.
    • CSS STYLES allow us to associate special rules for different HTML elements which effect how your overall page will look.
      • Ex: All <h1> tags will be red, <h2> tags will be blue, <p> tags will be green...

  • There are 2 ways to introduce CSS styling in an HTML page.
    1. The '<style>' tag is one place where we can introduce special styling rules.
      • Since we need these styles to appear right as the page loads, we introduce this tag in the <head> portion of our code.
    2. A 'CSS style sheet' a file separate from your HTML code.
      • By separating our styles from our HTML, we maintain a clean, easy to read structure. This is very helpful when it comes to debugging.
      • If we create this separate file, we need to use the <link> tag to let our browser know where to find our new rules!
        • (We will go over this later...)


where-do-we-start:

  • You may have noticed some weird code in the image below named 'EXTERNAL CSS FILE'. This was a real application of how we would create these "special rules."

  • Let's break down this structure..
    • In order to make our HTML appear unique, we need to associate each element with the various styles you want to use. for different HTML elements which effect how your overall page will look.
      • A CSS rule is made up of 2 parts:
        1. The 'selector' is the indicator for which element we are styling.
          • <h1>,<h2>,<p>, etc.
        2. The 'declaration' refers to the statements of how we want the selector to be styled.
          • font-size : 18px, font-color: blue, background-color: white, etc.
      • The 'declaration' of a style also has 2 parts!
        1. The 'property' of a declaration indicates what attribute of the style we are changing.
          • font-size, color, background-color, width, height etc.
        2. The 'value' specifies the setting of the desired property.
          • font size (18px), color (red), width (200px) etc.




link-up:

How to link an external CSS file



  • As mentioned earlier, when linking external files, we have to reference them in the <link> tag.
    • We need to include 4 attributes in this tag:
      1. <link>
        • This tells our browser: "Okay you need to look for something..."
      2. href="##filepath##"
        • This tells our browser: "Okay in order to make my site look awesome, look 'here'."
        • Typically the file path would look like this: 'styles/styles.css'.
          • styles = the folder storing the external file
            styles.css = the actual file we are looking for.
      3. type="text/css"
        • This tells our browser: "The type of file I am feeding you is a CSS file."
      4. rel="stylesheet"
        • This is the relationship between the HTML page and the file we are linking it to.
