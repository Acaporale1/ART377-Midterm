How do you navigate from page to page?



  • Whether you are shopping online or scrolling through social media, you most definitely have interacted with links.
    • When jumping from page to page, HTML uses the '<a>' tag.
      • This is called an 'anchor' tag.

  • Anything placed between the open <a> and closed </a> is clickable by the user.

  • In order to specify where we want the user to be redirected, we use the 'href="##URL##"' attribute.
    • The cool thing about anchor tags is that, opposed to the '<link>' tag we learned about last lesson, your 'href="##URL##"' here can contain a website URL (https:www.google.com), another page within the current site the user is on, or an email address! (index.html, contact.html, etc.)


<a href="https://google.com">CLICK HERE</a>

theres-more-to-links-that-meets-the-eye:

Links have more going on behind the scenes...



  • The hard part is out of the way. Now that we have our tag redirecting the user to another source, we need to add a few more things.

  • One thing to keep in mind when designing a website is the USER EXPERIENCE.
    • This means: 'Is it easy for a user to understand the objective of your website? Is it readable? Is it easy to navigate?

  • When embedding links in your webpage, think about these questions:
    1. When a user clicks this link, should they stay in the current browser window?
    2. When a user clicks this link, should a new browser window open?

  • To alter this experience, it is important to include the 'target="_"' attribute.
    • There most common targets are:
      1. target="_self"
        • By using this target, the link will redirect to another page in the SAME window.
      2. target="_blank"
        • By using this target, the link will redirect to another page in a NEW window.
        • If you take this approach, you may hope that the user will go back to the original window they were browsing on after they are finished with the embedded link.

  • One last habit you should get into is adding the ' title=" " ' attribute.
    • The title attribute acts as a TOOLTIP. In simpler terms, when you hover over a link, the text you include within the title attritube will appear in a small box to the right of your mouse cursor.
    • You may not have noticed, but when you hover over the CLICK HERE listed above, you see the text: 'GOOGLE.COM'.
      • The syntax for this link is as follows:
        <a href="https://www.google.com" target="_blank" title="GOOGLE.COM">CLICK HERE</a>
