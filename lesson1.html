understanding-the-structure:


  • HTML code is comprised of a series of characters that live inside angled brackets.
    • These angled brackets are considered your less-than and greater-than signs!

  • The official term for this combination of characters is an 'ELEMENT'!

  • These ELEMENTS are typically made up of two 'TAGS' :
    1. 'OPEN' tag.
    2. 'CLOSED' tag.

As we get more immersed into the material, you will begin to understand different types of HTML tags and how an internet browser interprets them!



getting-started:


It's important to understand the basic structure of an HTML page before we dive into the design...

  • An HTML page is structured like a human body!
    • Although this is not required, it is always a good idea to tell the browser what kind of document it is reading.
    • As we just learned, HTML uses TAGS.
    • To tell the browser that it is reading an HTML document, we use the tag :

      • <!DOCTYPE HTML>


  • Okay, we told the browser we want to code in HTML... but now what?
    • Back to our analogy, our whole document is going to contain HTML code.
      • We want to start with an open HTML tag.

      <html>


    • The head of a person contains the brain, one of the most essential components of the human body that processes information.
      • In HTML, we will give our code a 'brain' using the head tag!
      • When we start learning about linking CSS files, we will be using this tag more. For now just make sure you have an open head tag and a closed head tag.

      <head> </head>


    • Awesome! Now time for the body.
      • The body tag is where all the magic happens. Everything inside of the body tag is what is shown in the browser.
        • Think of this as where all of the human organs belong: our heart, liver, kidney, etc.
      • Since we are only going over the structure, we will just put an open body tag and a closed body tag.

      <body> </body>


    • We're almost done! We are just missing one tiny detail.
      • If you noticed, we had opened AND closed both the head and body tags... What about the html tag?
      • Like a human body, our code must have a top and bottom. When we are finished with our head and body , we wrap up our code but inserting a closed html tag on the last line of our document.

      </html>


Notice how in the figure above, none of the lines for matching elements are crossed. If you trace your code as you work, your code will remain error-free!
